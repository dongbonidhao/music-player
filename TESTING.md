# 🎵 音乐播放器音源测试指南

## 快速测试步骤

1. **打开 `index.html`**
2. **选择音源**（右上角下拉菜单）
3. **关闭代理**（先试不用代理）
4. **搜索歌曲**（如"周杰伦"）
5. **查看结果**：
   - 如果搜索成功 → 音源可用 ✅
   - 如果失败 → 开启代理再试
   - 还失败 → 换下一个音源

## 🔍 如果搜索成功但播放失败

这种情况说明：
- ✅ 搜索接口正常
- ❌ 播放接口需要代理或有其他限制

**解决方法：**
1. 开启代理（Codetabs 或 CORS Anywhere）
2. 再次尝试播放
